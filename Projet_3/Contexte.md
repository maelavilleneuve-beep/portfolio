# PROJET 3 - Requêtez une base de données avec SQL

## Résumé : 
Création et requêtage d'une base de données relationnelle avec SQL. Projet d'entraînement aux fondamentaux : schéma relationnel, chargement des données et scripts d'analyse pour la prise de décision.

## Compétences :
- Charger des données dans une base
- Créer un schéma de base de données
- Rédiger des scripts SQL pour la prise de décision

## Soft Skills :
- Rigueur méthodologique
- Organisation du travail
- Esprit de synthèse (restitution méthodologique)
