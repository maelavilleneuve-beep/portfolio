# PROJET 2 - Visualisez des données avec Excel

## Résumé :
Analyse de données sportives et création de visualisations accessibles dans Excel. Premier projet technique centré sur le choix des graphiques et les bonnes pratiques d'accessibilité (contrastes, titres, légendes).

## Compétences :
- Concevoir des graphiques adaptés
- Bonnes pratiques d'accessibilité

## Soft Skills
- Communication :
  - Communication orale
  - Argumentation et justification des choix
- Analyse :
  - Esprit de synthèse
  - Rigueur analytique
- Relation client :
  - Sens du client
