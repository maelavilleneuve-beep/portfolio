# PROJET 1 - Démarrez votre formation de Business Intelligence Analyst

## Résumé :
Prise en main du parcours de formation : définition de l'objectif professionnel, découverte du métier de BIA et identification de ses soft skills. Projet d'orientation sans livrable technique.

## Compétences :
 - Identifier les softs skills du métier de BIA
 - Définir le cadre de la formation
 - Développer la posture d'un BIA

# Soft Skills
- Autonomie :
  - Autonomie dans l'apprentissage
- Posture :
  - Connaissance de soi
  - Clarté des objectifs
  - Posture professionnelle
